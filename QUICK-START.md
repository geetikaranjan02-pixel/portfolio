# 🎯 Quick Start - 5 Minutes to Live

## The Files You Have

- `index.html` - Your portfolio page (don't edit)
- `style.css` - Styling (don't edit)
- `main.js` - Functionality (don't edit)
- `config.json` - **YOUR CONTENT** ← Edit this!
- `GITHUB-PAGES-SETUP.md` - Full deployment guide

---

## 3 Easy Steps

### 1️⃣ Edit Your Content

Open `config.json` and update:

```json
"name": "Your Name",
"email": "your@email.com",
"linkedin": "https://linkedin.com/in/yourprofile"
```

Add your projects, bio, skills, etc.

### 2️⃣ Push to GitHub

```bash
git remote add origin https://github.com/YOUR-USERNAME/portfolio.git
git branch -M main
git push -u origin main
```

(Replace `YOUR-USERNAME` with your GitHub username)

### 3️⃣ Enable GitHub Pages

1. Go to your repo on GitHub
2. Settings → Pages
3. Branch: `main` → Save
4. Wait 1-2 minutes
5. Your site is live! ✅
